## 🟡 중요 (이사 전까지)

- [ ] 이사 날짜 확정
- [ ] **[현재 전세집] 세탁기 분해 업체 컨택 및 예약** — 이사 전 반출 필요
- [ ] **[새 집] 문틀 분해 제거 업체 예약** — 세탁기 반입 위해 필요
  - 부동산에 전화해서 가능한 업체 번호 받기
  - 업체와 일정 사전 소통 (이사 날짜 확정 후 바로 예약)
- [ ] 인터넷 이전 신청 (이사 2주 전)
- [ ] 짐 정리 — 버릴 것 / 보관 / 이사 갈 것 분류
- [ ] 가전 이전 설치 예약
  - 에어컨 이전 설치
  - 세탁기 이전 설치 (문틀 제거 완료 후 가능)
- [ ] 구매 가전 주문 (식기세척기, 정수기, 로봇청소기)
- [ ] 현관 센서등 교체
- [ ] 무광 수전 교체 (원하는 경우)
- [ ] 도배 여부 결정 및 일정 수립
- [ ] 이사업체 견적 3곳 이상 받기 (성수기 — 최소 4주 전 예약)
- [ ] 인테리어/수리 항목 확정 후 업체 섭외
