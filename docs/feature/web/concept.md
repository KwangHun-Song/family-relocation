# 웹서비스 — 개발 컨셉

> 최종 업데이트: 2026-04-09

---

## 배경 / 동기

가족 이사 프로젝트의 모든 진행 상황과 남은 할 일이 로컬 마크다운 파일로 관리되고 있다.
현재는 파일을 직접 열거나 Claude Code를 통해서만 내용을 확인할 수 있어, 가족 구성원 누구나 언제 어디서든 현황을 확인할 수 있는 웹 뷰어가 필요하다.

---

## 핵심 목표

- 로컬 마크다운 파일을 웹에서 읽기 좋게 렌더링
- 가족만 접근할 수 있도록 Google OAuth 인증
- git push만으로 자동 반영 (별도 배포 작업 없음)
- 별도 DB 없음 — 마크다운 파일이 단일 진실 소스(Single Source of Truth)

---

## 핵심 기능

| 기능 | 설명 |
|------|------|
| 홈 대시보드 | 4개 카테고리 현황 요약 + 전체 투두리스트 미리보기 |
| 전체 투두리스트 | `전체-투두리스트.md` 렌더링 — 기간별 정렬 |
| 카테고리별 뷰 | 01~04 폴더의 `현재상황.md`, `투두리스트.md`, `진행사항.md` 렌더링 |
| Google OAuth 인증 | 허용된 이메일 주소 목록(`ALLOWED_EMAILS`)만 접근 가능 |
| 자동 배포 | git push → Vercel 자동 빌드 및 배포 |

---

## 대상 사용자

- 가족 구성원 (부부, 양가 부모님 등 허용된 이메일 소유자)
- 모바일 및 데스크탑 모두 사용 가능해야 함

---

## 범위 제외 (만들지 않는 것)

- 웹에서 마크다운 편집 기능 (읽기 전용)
- 댓글, 알림, 푸시 기능
- 별도 데이터베이스
- 관리자 페이지
